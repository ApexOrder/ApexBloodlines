# Installation

## Client Install

1. Download the latest Apex Bloodlines release zip.
2. Extract the `ApexBloodlines` folder.
3. Place it into your 7 Days to Die `Mods` folder.
4. Start the game.

Expected path:

```text
7 Days To Die/Mods/ApexBloodlines/
```

## Server Install

Install the same mod folder on the dedicated server.

If future versions include custom assets, clients may also need the mod installed locally.

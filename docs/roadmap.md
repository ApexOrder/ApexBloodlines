# Apex Bloodlines Roadmap

Apex Bloodlines is developed through named milestone releases. Each version has a clear theme, codename, and set of Epics.

## Release Codenames

| Version | Codename | Theme |
|---|---|---|
| Alpha 0.1 | Genesis | Project foundations |
| Alpha 0.2 | Awakening | Bloodline framework |
| Alpha 0.3 | Knowledge | Tome system |
| Alpha 0.4 | Cornerstone | Keystone framework |
| 1.0 | Foundations | First public release |
| 1.1 | Ascension | Bloodline specialisations |
| 1.2 | Industry | Advanced production |
| 1.3 | Machina | Engineering and vehicles |
| 1.4 | Harvest | Survival overhaul |
| 1.5 | Warpath | Combat overhaul |
| 1.6 | Frontier | Exploration overhaul |
| 1.7 | Stronghold | Base progression |
| 1.8 | Legacy | Apex end-game |
| 2.0 | Reclamation | World evolution |

# Alpha Roadmap

Alpha builds are internal development builds used to create and test the core systems before public release.

## Alpha 0.1 — Genesis

### EPIC 001 — Project Foundation

Stories:
- Repository structure
- XML mod structure
- Localisation framework
- Version management
- Debug configuration
- Internal documentation

### EPIC 002 — Branding Foundation

Stories:
- Logo direction
- Mod banner
- Icon style guide
- Loading screen direction
- Public presentation language

# Public Release Roadmap

## Version 1.0 — Foundations

Goal: deliver the first public version of Apex Bloodlines with the full core gameplay loop represented.

### EPIC 100 — Six Starting Bloodlines

Stories:
- Prospector Tier 1
- Craftsman Tier 1
- Engineer Tier 1
- Survivalist Tier 1
- Pathfinder Tier 1
- Vanguard Tier 1

Each Bloodline includes:
- Identity
- Starting progression
- Tome I
- Basic unlocks
- Initial crafting role
- Player-facing documentation

### EPIC 101 — Keystone Foundation

Stories:
- Camp tier
- Outpost tier
- Initial Keystone upgrade logic
- Technology branch display
- Tome slot rules
- Base progression rules

### EPIC 102 — Tome Foundation

Stories:
- Industry Tome I
- Construction Tome I
- Engineering Tome I
- Agriculture Tome I
- Combat Tome I
- Exploration Tome I

### EPIC 103 — Crafting Foundation

Stories:
- Early crafting rebalance
- Basic profession recipes
- Basic repair rules
- Early workstation progression

### EPIC 104 — Feedback Foundation

Stories:
- Public issue templates
- Public roadmap
- Installation guide
- Changelog format
- Feedback categories
